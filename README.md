# My Food Delivery App

A modern, responsive **food delivery web application** built with **React** and ** CSS**.  
This app provides a seamless user experience for browsing menus, adding items to the cart, and placing orders online.  

### Key Features
- Dynamic **Home**, **Cart**, and **Place Order** pages  
- Add, remove, and update items in the cart  
- View real-time order summary with prices  
- Fully responsive layout for **desktop and mobile**  
- Clean and modern UI with ** CSS**
